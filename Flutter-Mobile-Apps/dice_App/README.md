# Dice App

A simple Flutter app that rolls two dice with a tap of a button.

<img src="1_APP_ScreanShot.png" alt="Splash Screen" width="250"> <img src="2_APP_ScreanShot.png" alt="Roll Dice Screen" width="250">

## Features

- Tap "Roll Dice" to randomly roll two dice
- Simple and colorful UI

## Getting Started

1. Clone or download this project.
2. Run `flutter pub get` to install dependencies.
3. Run `flutter run` to launch the app.

## Requirements

- Flutter SDK
- Dart
